# zakura-checkpoints

`zakura-checkpoints` uses a local `zakurad` or `zcashd` instance to generate a list of checkpoints for Zakura's checkpoint verifier.

Developers should run this tool every few months to add new checkpoints to Zakura.
(By default, Zakura uses these checkpoints to sync to the chain tip.)

For more information on how to run this program visit [Zakura checkpoints README](https://github.com/zakura-core/zakura/tree/main/zakura-chain/src/parameters/checkpoint/README.md)
