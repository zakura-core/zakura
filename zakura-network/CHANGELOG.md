# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Changed

- Temporary ordered-service demand refusals are now re-checked with bounded
  backoff while the underlying connection remains healthy.

### Fixed

- Block-sync no-progress liveness now parks only the local block-sync session
  and re-admits the peer after its cooldown without requiring a transport
  redial.

## [1.0.0] - 2026-07-15

First "stable" release. However, be advised that the API may still greatly
change so major version bumps can be common.

## Pre-fork history

This crate was forked from Zebra at v5.0.0. Earlier history is available in the
[upstream changelog](https://github.com/ZcashFoundation/zebra/blob/v5.0.0/zebra-network/CHANGELOG.md).
